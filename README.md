
 Hardest Part 

The most challenging part of the project was cleaning the funding data, especially handling missing values.  
Some records in the `amount_cad` column had no values, so I had to deal with them carefully to avoid losing important data.

To solve this, I replaced the missing values with the median value of the funding amounts.
